## Criar Personagem

2. Selecione uma [Classe e Arquétipo](classes.md)

## Sistema de Jogo

Auramana utiliza um sistema baseado em:

- D20 para testes de perícia e atributo
- Pontos de Vida e Mana
- Perícias especializadas
